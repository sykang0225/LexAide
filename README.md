# LexAide — 다국어 금융광고 준법심의 조력 AI

> **JB금융그룹 Fin:AI Challenge 2026 · 지정주제2 (Compliance AI) · 팀 JBIG 9th**

금융광고의 위반 가능성을 넓게 탐지하고, 번역본의 **규제 드리프트**까지 교차검증하는 준법심의 조력 AI Agent입니다.
핵심 원칙은 하나 — **위반은 넓게 탐지하되, 최종 판단은 사람(준법관리자)이 합니다.**

---

## 문제의식

준법 심의는 수작업 중심이라 다국어·다채널로 확장될수록 검토 부담과 **미탐(위반 누락) 위험**이 커집니다.
특히 한국어 원본을 인도네시아어로 번역하는 과정에서 위험 고지 누락 등 **규제 드리프트**가 발생합니다.
준법에서 위반 누락은 제재·과징금으로 직결되고 복구가 어렵기 때문에, **재현율(Recall) 우선**으로 설계했습니다.

## 핵심 기능

- **4-Agent 파이프라인** — ①언어·관할 라우팅 ②Rule→Embedding→LLM 3단 위반 탐지 ③번역 정합성 교차검증 ④인용 조문 실존 검증(환각 차단)
- **Rules as Code** — 법령 → 위반유형 → 판정규칙으로 내려가는 계층 트리(YAML). 조문 직결 위반은 규칙이 자동 판정, 애매한 표현은 LLM이 잡되 사람 검토로.
- **한·인니 이중관할 교차검증** — 원문(금소법)·번역본(OJK)을 독립 심의 후 판정 등급 비교 → 번역에서 새로 생긴 위반 탐지. 한·인니 용어집(Termbase)으로 오번역 수정 권고.
- **현저성·부작위 검사** — 불리 고지가 작게·하단·흐리게 표시된 비대칭 노출을 크기·위치·굵기·색상 **4축**으로 탐지.
- **노코드 규칙 관리** — 준법 전문가가 코드 수정 없이 규칙·법령을 직접 추가·조정. 인용 조문은 국가법령정보 API로 **실측 확인**(삭제·부존재 조문 저장 차단), 원본 법령 규칙은 보호(조정·중지만), 모든 변경은 **감사 추적** 자동 기록.
- **규제 변경 모니터링** — 트리 근거 법령의 공포·시행일자를 국가법령정보 API로 대조해 개정 감지(감지=자동, 대응=전문가).
- **심의 리포트 자동 출력** — 판정·근거 조문·수정 권고·결재 서식을 담은 A4 리포트. 화면의 승인·수정·반려 결정이 결재란에 자동 반영(HITL).

## 기술 스택

FastAPI · YAML DSL · FAISS · paraphrase-multilingual-MiniLM-L12-v2 · Groq Llama 3.3 70B · (선택) Anthropic Claude 비전 OCR · 국가법령정보 Open API

## 프로젝트 구조

```
agents/   4개 AI Agent (언어·관할 / 위반탐지 / 번역정합성 / 조문검증)
api/      FastAPI 서버 · 법령 API · 규제 모니터링 · 심의 이력
core/     규칙 트리 엔진 · 안전 편집 레이어(rule_store)
data/     법령 트리(YAML) · 평가셋 · 테스트 광고 · 용어집
static/   웹 준법 심의 콘솔 (index.html · app.js · style.css)
utils/    텍스트 추출 · 임베딩 · OCR · 현저성(layout_risk) 분석
scripts/  평가 · 산출물 생성
tests/    규칙 편집 · 피드백 루프 테스트
```

## 실행 방법

1. **의존성 설치** (Python 3.12+ 권장)
   ```bash
   pip install -r requirements.txt
   ```
2. **환경변수 설정** — `.env.example`를 `.env`로 복사 후 키 입력
   ```bash
   copy .env.example .env    # (Windows)   /   cp .env.example .env  (bash)
   ```
   - `GROQ_API_KEY` — 런타임 판정 LLM
   - `LAW_API_KEY` — 국가법령정보 Open API OC (law.go.kr 발급)
   - `ANTHROPIC_API_KEY` — (선택) 이미지 OCR·성능 baseline 비교
3. **서버 실행**
   ```bash
   run_server.bat          # (Windows)
   # 또는
   uvicorn api.server:app --host 127.0.0.1 --port 8000
   ```
4. 브라우저에서 **http://127.0.0.1:8000** 접속 → 한국어 심의 / 인니어 심의 / 교차검증 탭

> 데모용 광고 샘플은 `data/test_ads/`에 있습니다(한국어·인니어 단일 심의, 한·인니 교차검증 쌍).

## 설계 원칙

- **Recall 우선** — 위반 누락 비용 > 과탐 비용. PASS 임계값을 낮춰 넓게 탐지.
- **Human-in-the-loop** — AI는 위험 후보를 넓게 제시하는 Regulatory Radar. 승인·수정·반려는 항상 사람.
- **거버넌스** — 원본 법령 규칙은 삭제·구조 수정 불가(조정·적용 중지만), 전문가 추가 규칙만 삭제. 모든 변경은 사유·인용 조문 강제 + 이전값→이후값 감사 추적.
